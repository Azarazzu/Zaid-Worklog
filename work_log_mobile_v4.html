Connect Google Sheets

One-time setup — takes about 5 minutes.

1
Create a Google Sheet
Go to sheets.google.com → create a new blank sheet → name it Work Log Data.
2
Open Apps Script
In your sheet, click Extensions → Apps Script. Delete any existing code. Paste the entire contents of Code.gs (provided separately). Click Save.
3
Deploy as Web App
Click Deploy → New Deployment → Select type: Web app → Set "Who has access" to Anyone → Click Deploy → Copy the URL shown.
4
Paste your Web App URL below
Paste the deployment URL from Step 3 here and save.
